# Basic Calculator UI

A minimal calculator application with a graphical calculator UI and working keyboard shortcuts.

Utilising an external CSS and Js page to save eating up the page and facilitating using an editor for the JS and CSS code directly.

---

## Run with UV

1. First time only: In a terminal cd to directory / open folder in VS code > open new terminal

    ```bash
    uv init
    ```

2. Each time: Run streamlit app via uv

    ```bash
    uv run streamlit run calculator.py
    ```
